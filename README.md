# SimpleCustomLauncher

シンプルで使いやすいAndroidホームアプリ。

## Features

- カスタマイズ可能なショートカット配置
- 連絡先から電話・SMSショートカットを作成
- カレンダー・メモ帳などの内蔵機能
- ダーク/ライトテーマ対応
- バックアップ・復元機能
